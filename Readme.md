### Summary

- FastAPI application
- Combination of FizzBuzz logic and mock data from JSONPlaceholder

### Environment
- Python 3.7

### How to Run

- on a console, type
`unicorn main:app --reload`

- then on web browser, go to 
`127.0.0.1:8000/fizzbuzz/_number_`

- type in an integer 1-10 or 15 in place of `__number__`

### How to test 
- on a console, run
`pytest`
